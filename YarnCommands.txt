======================================
Yarn Commands created for this project
======================================

See "InventoryItems_DT" for Inventory item details. "ItemName" is the RowName for entries, NOT DISPLAY NAME.

Characters inheriting from the base ThirdPersonCharacter_BP have a "YarnCharacterName" String Variable, used to identify them in these functions/commands.

The Camera presenter will snap directly to the speaking character based on the defined Speaker Name in Yarn code.

Inventory Functions
==================
CheckInventory("YarnCharacterName","ItemName") - returns BOOL - Checks specified character's inventory for specific item, returns TRUE if found.
CheckStackCount("YarnCharacterName","ItemName") - returns INT - Checks specified character's inventory for specific item, returns stack size (0 if not found).
TryTradeItem("FromCharacter","ToCharacter","ItemName","StackCount") - returns BOOL - Attempts a trade of item and returns FALSE if it fails, true if it succceeds (partial trades return TRUE)
TryAddItem("YarnCharacterName","ItemName","StackCount") - returns BOOL - Attempts to add item and returns FALSE if it fails, TRUE if it succeeds
TryRemoveItem("YarnCharacterName","ItemName","StackCount") - returns BOOL - Attempts to remove item and returns FALSE if it fails, TRUE if it succeeds

Inventory Commands
=================
TradeItem("FromCharacter","ToCharacter","ItemName","StackCount") - Removes specified item/stack from FROM Character's Inventory and adds it to TO Character's Inventory
AddItem("YarnCharacterName","ItemName","StackCount") - Adds specified item/stack to specified character's inventory
RemoveItem("YarnCharacterName","ItemName","StackCount") - Removes specified item/stack from specified character's inventory

Action Commands
==============
Action("YarnCharacterName","ActionName") - Causes the specified character to run the Montage specified on the "ActionName" row of the Data Table YarnCharacterActions_DT.
